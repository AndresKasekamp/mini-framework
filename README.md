## 📚 Introduction

Your framework should implement:

- Abstracting the DOM Routing System
- State Management
- Event Handling
  
You will also need to make a todoMVC project using your framework.

## 👟 Requirements to run

- Bash terminal window
- Web browser

## 🏃‍♂️ Running the program

### ✈️ Terminal

1. Install vite

```bash
npm i
```

2. Run dev server

```bash
npm run dev
```
